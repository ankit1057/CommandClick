<strong>Command Click</strong> allow you change shellscript to Gui App.

<strong><font color="#339933">[Feature]</font></strong>

* Allow you change shellscript to Gui App
* One click execution for Shellscript
* Versatile usage for Terminal, Crome, OS setting, etc..
* Not only termux gui client but also web browser..
* Offer ritch edit dialog to termux.

<strong><font color="#339933">[Objective]</font></strong>

* Execute shellscript by one click.
* Handling termux command line by GUI.
* Self made Original browser by shellscript.
